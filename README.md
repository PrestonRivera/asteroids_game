# Asteroids game

## Play my version of the infamouse Asteroids game using PYGAME

### Installation 

1. Clone the repo
> https://github.com/PrestonRivera/asteroids_game.git
2. Go into the project directory
> cd user_path/asteroids_game
3. Create a virtual enviroment
> python -m venv venv
4. Activate the virtual enviroment
> source venv/bin/activate
5. Install the dependicies
> pip install -r requirements.txt
6. Run the game
> python3 main.py
